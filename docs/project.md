# Project Overview

## Product

Booking Mini App — система онлайн-записи на услуги для бизнеса, работающего через Telegram.

Продукт состоит из трех основных компонентов:

- **Web Admin** — веб-приложение для управления бизнесом
- **Telegram Mini App** — клиентское приложение внутри Telegram
- **Backend API** — сервер бизнес-логики
- **Telegram Bot** — бот для уведомлений и команд

Система позволяет бизнесам управлять услугами, сотрудниками и расписанием, а клиентам — записываться на услуги напрямую внутри Telegram.

---

## Основные роли

1. **Клиент**
   - записывается на услуги
   - управляет своими записями

2. **Владелец бизнеса / администратор**
   - управляет услугами
   - управляет сотрудниками
   - управляет расписанием
   - управляет записями клиентов

3. **Сотрудник (мастер)**
   - получает уведомления о новых записях
   - может просматривать своё расписание через Telegram Bot

---

## Основные возможности

### Для бизнеса (Web Admin)

- настройка информации о компании
- создание категорий услуг
- создание и редактирование услуг
- управление сотрудниками
- настройка графика работы сотрудников
- управление календарем записей
- создание записи вручную
- перенос и отмена записей
- просмотр базы клиентов

---

### Для клиентов (Telegram Mini App)

- просмотр информации о компании
- выбор услуги с поиском и фильтрами (категория, специалист)
- выбор даты и времени (bottom sheet)
- ввод контактных данных (имя, фамилия, телефон, комментарий)
- подтверждение записи (согласие на обработку ПД)
- просмотр своих записей (вкладка «Записи»)
- профиль (вкладка «Профиль»)
- перенос или отмена записи

---

### Через Telegram Bot

- уведомления мастеру о новых записях
- напоминания клиенту о записи
- кнопки быстрого перехода к записи
- команды просмотра расписания мастера

Примеры команд:

- `/today`
- `/tomorrow`
- `/week`

---

## Основные сущности

Основные доменные сущности системы:

- **User** — пользователь системы
- **Business** — бизнес/организация
- **ServiceCategory** — категория услуг
- **Service** — услуга
- **Employee** — сотрудник (мастер)
- **Schedule** — график работы сотрудника
- **Booking** — запись клиента
- **Client** — клиент бизнеса
- **TimeSlot** — доступный временной слот
