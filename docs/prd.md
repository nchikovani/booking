# Product Requirements Document (PRD)
## Telegram Mini App — Booking System

---

# 1. Общий обзор продукта

## Краткое описание
**Booking Mini App** — это Telegram Mini App, позволяющий бизнесам принимать онлайн-записи на услуги прямо внутри Telegram.

Клиенты могут:

- выбрать услугу
- выбрать дату и время
- забронировать слот
- получить напоминание

Владельцы бизнеса получают:

- календарь записей
- управление услугами
- управление расписанием
- уведомления

Приложение работает как **mini CRM + booking система внутри Telegram**.

---

## Какую проблему решает продукт

Многие малые бизнесы принимают записи вручную через чат.

Типичный процесс:

1. клиент пишет  
2. бизнес отвечает  
3. согласовывается время  
4. запись фиксируется вручную  

Проблемы:

- потерянные сообщения  
- двойные бронирования  
- много ручной работы  

Booking Mini App автоматизирует процесс.

# 5. Функциональные требования

---

## Создание услуги

Описание  
Владелец бизнеса может создать новую услугу.

Trigger  
Нажатие кнопки **Add Service**

Result  
Услуга сохраняется и отображается в списке услуг.

---

## Создание записи

Описание  
Клиент выбирает услугу и свободный слот.

Trigger  
Нажатие кнопки **Confirm booking**

Result  
Создается запись в календаре.

---

## Отмена записи

Описание  
Пользователь может отменить запись.

Trigger  
Нажатие кнопки **Cancel**

Result  
Запись удаляется, слот освобождается.

---

## Напоминания

Описание  
Система отправляет напоминания перед записью.

Trigger  
Приближение времени записи

Result  
Клиент получает уведомление в Telegram.

# 9. Состояния приложения

---

## Новый пользователь

- нет услуг
- нет записей

UI показывает onboarding.

---

## Активный пользователь

Есть:

- услуги
- записи
- история бронирований

---

## Пользователь без данных

Нет:

- услуг
- клиентов

UI показывает подсказки и onboarding.

---

## Ошибки

- сеть недоступна
- серверная ошибка
- ошибка записи

UI отображает сообщение об ошибке.

---

## Ограничения

Free план:

- 1 календарь
- до 50 записей в месяц
